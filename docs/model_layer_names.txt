This model was trained by Sergio Guadarrama @sguada

Places205-GoogLeNet model
http://places.csail.mit.edu/downloadCNN.html
GoogLeNet CNN trained on 205 scene categories of MIT Places Database with  2.5 million images
Please cite this work if you use the database or CNN:
	B. Zhou, A. Lapedriza, J. Xiao, A. Torralba, and A. Oliva. “Learning Deep Features for Scene Recognition using Places Database.” Advances in Neural Information Processing Systems 27 (NIPS), 2014. PDF Supplementary Materials
This model is released for unrestricted use.
